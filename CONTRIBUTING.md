# Contributing

Contribuicoes sao bem-vindas.

## Fluxo recomendado

1. Faca um fork do repositorio
2. Crie uma branch
3. Rode os checks
4. Abra um pull request

```bash
npm install
npm run check
npm run build
```

## Padrao

- TypeScript strict
- Codigo simples e direto
- Sem credenciais em commits
- Testar backup manual antes de alterar scheduler
